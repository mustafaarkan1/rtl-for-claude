# Privacy Policy for RTL for Claude

This extension does not collect, store, or transmit any user data. 

### Data Usage
- **No Data Collection:** We do not collect any personal information, browsing history, or chat content.
- **Local Execution:** All script injections and CSS modifications happen locally within your browser on the `claude.ai` domain.
- **No Third-Party Sharing:** Since no data is collected, no data is shared with or sold to third parties.

### Contact
If you have any questions, you can open an issue on this GitHub repository.
